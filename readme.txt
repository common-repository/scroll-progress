== Description ==

Displays a horizontal progress bar that shows a scroll position relative to the height of the opened page. Useful for websites with longer articles. Developed by worbee.com

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
